## تشغيل البرنامج / How to Run

### العربية

1. اضغط على زر **Code** ثم **Download ZIP**
2. فك الضغط عن الملف
3. دبل كليك على **setup.bat**
4. البرنامج هيثبت كل المتطلبات ويشتغل تلقائياً

> ملاحظة: لو ما عندكش Python، الـ setup هيطلعلك رسالة فيها لينك التحميل.

---

### English

1. Click **Code** then **Download ZIP**
2. Extract the ZIP file
3. Double-click **setup.bat**
4. The setup will install all requirements and launch the app automatically

> Note: If Python is not installed, the setup will show you a message with the download link.

## المميزات / Features

**بالعربية:**
- اضافة بيانات الموظفين كاملة (اسم، وظيفة، قسم، سن، رقم قومي، عنوان، تليفون، حالة اجتماعية، رقم طوارئ)
- توليد رقم موظف تلقائي بالصيغة ECN-ENG-001 / ECN-TECH-001 / ECN-ADM-001
- حفظ البيانات في ملف Excel منسّق
- تحميل ملف الموظفين من Google Sheets او Google Drive مباشرة
- انشاء ملف Excel جديد او فتح ملف موجود

**English:**
- Add full employee records (name, job title, department, age, national ID, address, phone, marital status, emergency contact)
- Auto-generate employee ID in the format ECN-ENG-001 / ECN-TECH-001 / ECN-ADM-001
- Save data to a formatted Excel file
- Load employee file directly from Google Sheets or Google Drive
- Create a new Excel file or open an existing one
